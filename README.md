# Tugas Akhir Membuat Website - Dicoding Submission

Ini adalah proyek submission untuk kelas **Belajar Dasar Pemrograman Web** dari Dicoding. Proyek ini merupakan sebuah website sederhana yang menampilkan artikel dan biodata penulis, dengan layout yang disusun menggunakan **Flexbox** sesuai kriteria penilaian.

## Fitur Utama
- ✅ Elemen semantik HTML5: `<header>`, `<nav>`, `<main>`, `<article>`, `<aside>`, `<footer>`
- ✅ Navigasi dengan tautan ke profil Dicoding (sesuai ketentuan)
- ✅ Foto profil di dalam elemen `<aside>`
- ✅ Layout responsif menggunakan **Flexbox** (tanpa float)
- ✅ Tampilan bersih dan mudah dibaca

## Built With
- **HTML5** - Struktur halaman
- **CSS3** - Styling dan layout (Flexbox)
